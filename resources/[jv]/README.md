# Jv420 resources

## jv-loading
Simpele loading screen.

## jv-discordlogs
Join/leave logs naar Discord webhook.

## jv-notify
Commands:

- `/welkom`
- `/regels`

## jv-adminhelp
Commands:

- `/jvhelp`
- `/jvplayers`
- `/jvannounce tekst`

Zet deze in `server.cfg`:

```cfg
ensure jv-loading
ensure jv-discordlogs
ensure jv-notify
ensure jv-adminhelp
```
