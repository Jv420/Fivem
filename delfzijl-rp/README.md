# Delfzijl RP - FiveM ESX Legacy Pack

Dit is een starter-pack voor een Nederlandse ESX Legacy FiveM server.

## Wat zit erin?
- server.cfg voor Nitrado/txAdmin
- SQL starter database additions
- Nederlandse Delfzijl resources
- OX/ESX startvolgorde
- gratis script-downloadlijst
- Nederlandse RP instellingen
- jobs/licenties/illegale RP templates

## Belangrijk
Externe scripts zoals ESX Legacy, ox_lib, oxmysql, ox_inventory en ox_target zitten niet meegebundeld. Download die via de officiële GitHub releases en plaats ze in de juiste map.

## Installatie kort
1. Upload deze map naar je FiveM server.
2. Download ESX Legacy + OX files.
3. Plaats ze in resources/[ox] en resources/[esx].
4. Importeer sql/delfzijl_starter.sql.
5. Vul je database string in server.cfg in.
6. Start via txAdmin.
