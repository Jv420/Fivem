# Items toevoegen aan ox_inventory

Open `ox_inventory/data/items.lua` en voeg toe:

```lua
['weed_leaf'] = {
    label = 'Wietblad',
    weight = 10,
    stack = true,
    close = true,
},
['weed_bag'] = {
    label = 'Zakje wiet',
    weight = 20,
    stack = true,
    close = true,
},
['coca_leaf'] = {
    label = 'Cocablad',
    weight = 10,
    stack = true,
    close = true,
},
['coke_bag'] = {
    label = 'Zakje coke',
    weight = 20,
    stack = true,
    close = true,
},
```
