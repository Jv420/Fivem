# Veiligheid en performance

- Gebruik alleen officiële releases of betrouwbare GitHub bronnen.
- Zet geen database wachtwoord openbaar op GitHub.
- Gebruik txAdmin permissions voor staff.
- Test nieuwe scripts eerst op een testserver.
- Vermijd leaked/pirated scripts; die bevatten vaak backdoors.
- Gebruik geen scripts met obfuscated server code tenzij je de maker vertrouwt.
