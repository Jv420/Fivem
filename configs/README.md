# Config templates

Deze map bevat planning en voorbeeldinstellingen voor je FiveM server.

## Mappen

- `economy` - geldbalans en prijzen
- `jobs` - jobs planning
- `shops` - shop planning
- `police` - politie setup
- `ems` - ambulance setup
- `mechanic` - mechanic setup
- `qb-core` - qb-core voorbeeldinstellingen

Let op: de meeste bestanden zijn templates/plannen. Kopieer niet blind over bestaande QBCore configs zonder backup.
