# Discord setup voor FiveM RP

## Rollen

- Owner
- Admin
- Moderator
- Support
- Developer
- Politie
- Ambulance
- Mechanic
- Burger
- Donateur

## Kanalen

### Informatie

- welkom
- regels
- aankondigingen
- server-status
- changelog

### Community

- algemeen
- screenshots
- clips
- suggesties
- vragen

### Support

- ticket-aanmaken
- reports
- bug-reports

### Logs privé

- join-leave-logs
- admin-logs
- donation-logs
- error-logs

## Webhooks

Maak minimaal deze webhooks:

- Join/leave logs
- Staff logs
- Donation logs
- Error logs

Zet de webhook in:

```txt
resources/[jv]/jv-discordlogs/config.lua
```
