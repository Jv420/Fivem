# Gratis FiveM scripts en mods checklist

Deze lijst is bedoeld als veilige start. Download alleen van officiële GitHub-pagina's, Cfx.re releases of bekende makers.

## Core stack

| Type | Aanrader | Waarvoor |
|---|---|---|
| Framework | QBCore | Basis roleplay/economy framework |
| Database | oxmysql | MySQL connectie |
| Library | ox_lib | UI, callbacks, helpers |
| Target | ox_target | Interacties met NPC's/deuren/objecten |
| Inventory | ox_inventory | Moderne inventory |

## Wat niet gebruiken

Gebruik geen:

- Leaked scripts
- Nulled paid scripts
- Dubieuze Discord downloads
- Scripts met obfuscated server-side code zonder reputatie
- Resources die vragen om rare externe keys zonder uitleg

## Veilige installatievolgorde

1. FXServer artifacts
2. txAdmin recipe of handmatige setup
3. Database aanmaken
4. oxmysql installeren
5. qb-core installeren
6. QBCore dependencies installeren
7. Jobs/resources één voor één toevoegen
8. Server console checken op errors
9. Pas daarna extra auto's/maps/kleding toevoegen
