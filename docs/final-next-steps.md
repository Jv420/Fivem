# Wat je nu moet doen

1. Download deze repo als ZIP of clone hem.
2. Zet je FXServer in `C:\FXServer\server`.
3. Run `tools/setup-folders.bat`.
4. Kopieer `resources/[jv]` naar `C:\FXServer\server\resources\[jv]`.
5. Kopieer `server.cfg.example` naar `C:\FXServer\server\server.cfg`.
6. Vul je license key in.
7. Vul je database wachtwoord in.
8. Installeer QBCore + ox resources.
9. Importeer database SQL.
10. Start met `start-server.bat`.
