# Eerste start checklist

## Voor start

- [ ] FXServer.exe staat in je servermap
- [ ] `server.cfg` bestaat
- [ ] `sv_licenseKey` is ingevuld
- [ ] MySQL/MariaDB draait
- [ ] Database `fivem_qbcore` bestaat
- [ ] `mysql_connection_string` klopt
- [ ] `oxmysql` staat in resources
- [ ] `qb-core` staat in resources
- [ ] Eigen `[jv]` map staat in resources

## Tijdens start

Let op deze errors:

- `Could not find dependency`
- `No such export`
- `Access denied`
- `Failed to connect to database`
- `Could not open resource metadata file`

## Na start

- [ ] Server zichtbaar in Direct Connect
- [ ] Join werkt
- [ ] Loading screen werkt
- [ ] Character menu opent
- [ ] Spawn werkt
- [ ] Database maakt players aan
- [ ] `/welkom` werkt
- [ ] `/regels` werkt
- [ ] Console command `jvlogtest` werkt
- [ ] Console command `jvplayers` werkt
