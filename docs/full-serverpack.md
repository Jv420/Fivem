# Jv420 Full Starter Serverpack

Deze repo bevat nu een uitgebreidere eigen starterpack bovenop QBCore/ox.

## Eigen Jv420 resources

- `jv-loading` - loading screen
- `jv-discordlogs` - Discord join/leave logs
- `jv-notify` - `/welkom` en `/regels`
- `jv-adminhelp` - helper/admin commands
- `jv-hud` - simpele HUD met ID/HP/armor/straat
- `jv-bank` - `/bank`, `/deposit`, `/withdraw`
- `jv-starterjobs` - vuilnisman, taxi, bezorger basis
- `jv-weaponshop` - veilige starter weaponshop commands
- `jv-garage` - garage waypoint helper

## Nog officieel downloaden

Deze externe resources moet je zelf officieel downloaden:

- `qb-core`
- `qb-hud` optioneel, als je liever QBCore HUD gebruikt
- `qb-banking` optioneel, als je liever QBCore banking gebruikt
- `qb-policejob`
- `qb-ambulancejob`
- `qb-shops`
- `qb-garages`
- `oxmysql`
- `ox_lib`
- `ox_inventory`
- `ox_target`

## Commands

```txt
/hud
/bank
/deposit bedrag
/withdraw bedrag
/jobs
/startjob garbage
/startjob taxi
/startjob delivery
/finishjob
/weaponshop
/buyweapon flashlight
/buyweapon pistolammo
/garages

/setgarage sandy
/setgarage paleto
```

## Let op

De Jv420 resources zijn starterversies. Ze zijn bedoeld om snel te testen en later uit te breiden met echte UI's, target-zones, NPC's en database-opslag.
