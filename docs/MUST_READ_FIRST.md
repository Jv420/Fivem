# Eerst lezen

Topper, begin hiermee voordat je scripts installeert.

## Wat is deze repo?

Dit is een veilige starterbasis voor je FiveM server op Windows. De repo bevat eigen Jv420 resources, setupbestanden, documentatie, templates en database-voorbereiding.

De grote externe resources zoals QBCore, oxmysql, ox_lib, ox_target en ox_inventory moet je zelf downloaden via de officiële bronnen. Zo voorkom je oude bestanden, malware, leaked scripts en licentieproblemen.

## Startvolgorde

1. FXServer werkend krijgen.
2. Database werkend krijgen.
3. QBCore en ox installeren.
4. Jv420 resources toevoegen.
5. Daarna pas extra jobs, maps, auto's en kleding.

## Belangrijk

Zet nooit echte secrets in GitHub:

- FiveM license key
- Database wachtwoord
- Discord webhook
- API keys

Gebruik `server.cfg.example` als voorbeeld en maak lokaal je echte `server.cfg`.
