# Serverregels NL

## Algemene regels

1. Respecteer alle spelers en staffleden.
2. Geen racisme, haat, bedreigingen of extreem toxic gedrag.
3. Geen cheats, executors, mod menus of exploits.
4. Geen bug abuse. Meld bugs bij staff.
5. Geen reclame voor andere servers.

## Roleplay regels

1. Blijf zoveel mogelijk in character.
2. Geen random deathmatch.
3. Geen vehicle deathmatch.
4. Geen combat logging.
5. Geen powergaming.
6. Geen metagaming.
7. FearRP is verplicht in serieuze situaties.

## Criminaliteit

1. Robberies mogen alleen met voldoende politie online.
2. Geen campen bij ziekenhuizen, garages of spawn.
3. Gangs moeten roleplay opbouwen.
4. Hostages moeten echte spelers zijn, geen vrienden die nep meewerken.
