# Roadmap Economy/Fun RP

## Fase 1 - Playable Alpha

Doel: jij en testers kunnen joinen en basis gameplay testen.

Taken:

- FXServer + txAdmin installeren
- MariaDB installeren
- QBCore installeren
- ox stack installeren
- Jv420 starter resources testen
- Loading screen testen
- HUD testen
- Bank testen
- Starter jobs testen
- Vehicle helper testen

Klaar als:

- Speler kan joinen
- Character flow werkt
- Geld werkt
- Minimaal 2 jobs werken
- Geen grote console errors
